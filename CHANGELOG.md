# Changelog

## 0.1.0 - 2026-06-08

- Initial public-ready release of `review-benchmark-harness`
- Added offline CLI commands: `score`, `init-suite`, `normalize-output`, `compare`, `check`
- Added unified diff parsing and touched-line extraction
- Added JSON and Markdown review output normalization
- Added fuzzy file and line matching with severity mapping
- Added precision, recall, F1, rule coverage, severity accuracy, and localization metrics
- Added Markdown, JSON, CSV, and JUnit reporting
- Added baseline comparison and leaderboard output
- Added sample benchmark suite and CLI smoke coverage
- Added GitHub Actions CI and 28+ unit tests
